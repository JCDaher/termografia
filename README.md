# Termografia - Análise de Dermátomos e BTT

Aplicativo Windows desktop para análise de imagens termográficas FLIR radiométricas.

## Funcionalidades

- 🔥 Processamento de imagens FLIR radiométricas
- 📊 Análise de assimetrias térmicas em dermátomos
- 🧠 Análise BTT (Brain Thermal Tunnel)
- 🤖 Geração de laudos com Claude AI
- 📄 Exportação em PDF profissional

## Instalação
```bash
# Clonar repositório
git clone https://github.com/JCDaher/termografia.git
cd termografia

# Criar ambiente virtual
python -m venv venv
venv\Scripts\activate

# Instalar dependências
pip install -r requirements.txt

# Executar
python main.py
```
